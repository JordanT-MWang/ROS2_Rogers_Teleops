# ROS2_Rogers_Teleops
Robohub repository for the gravity and spring balanced haptic devices  
Developed a fully complete force feedback haptic system. Features included wireless haptics in 5G, Wi-Fi, Ethernet, and simulated delay for a research project between the University of Waterloo and Rogers Communication.  
  
Skills: C++, Python, Control Systems, ROS2, Circuit Design, Soldering  
![Untitled video - Made with Clipchamp (1)](https://github.com/user-attachments/assets/fc52495f-427f-440a-8b68-52a30c80e6df)


What?

- Created a Bilateral haptic teleoperation system using ROS2, Python, and C++ that utilized 5G, Wi-Fi, and Ethernet communication methods.

Why?

- Gain an understanding of how the latency of different networks affects the stability and usability of bilateral haptic teleoperation systems.
